## 🎮 My Game Project – Lioren

Lioren is my first completed game project developed in Unreal Engine.

It is an action platformer where players explore a cursed keep, unlock new forms and abilities, and fight powerful enemies.

### Features
- Multiple forms with unique abilities
- Upgradeable and unlockable skill system
- Combat and boss fights
- Exploration and progression
- Save system

🔗 Play here:  
https://taisui53.itch.io/lioren-thesealedpath

---

### Development

This project was developed over approximately 5 months.

During this time, I focused on building core gameplay systems such as abilities, combat, progression, and player state management using Unreal Engine Blueprint.

It gave me hands-on experience in designing gameplay systems, structuring logic, and completing a full game project from start to finish.

---

### Key Learnings

- Designing modular gameplay systems
- Managing player progression and ability systems
- Building and completing a full game project
- Iterating on gameplay and player experience
